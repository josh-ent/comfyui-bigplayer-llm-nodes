# BigPlayer Prompt (Split)

Converts freeform prose into split positive and negative prompt channels plus comments.

Inputs:
- `prose`
- `api_key`
- `llm_model`
- `MODEL`
- `style_policy`
- `provider_base_url`
- `assume_determinism`

Outputs:
- `text_l_positive`
- `text_g_positive`
- `text_l_negative`
- `text_g_negative`
- `comments`
